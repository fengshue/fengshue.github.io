CSS三大特性

继承性、层叠性、优先级
继承性：子元素可以继承父元素的样式。
什么样的属性才可以继承：
color，以text-,font-,line-开头的属性都是可以继承的

层叠性：是浏览器处理冲突的一个特性。
如果一个属性通过两个选择器设置到同一个元素上面，那么这个时候一个属性就会将另一个属性层叠掉。
如果多个不同的属性通过两个选择器设置到一个元素上面，那么不会发生层叠。

层模型

都在出生位置展示，但设置left、right值就按照规则定位
1:绝对定位
脱离原来位置进行定位
相对于有定位父级定位，没有的话相对于文档定位
不定义left等值，默认值为auto
position: absolute;
left:100px;
top:100px;

2:相对定位
保留原来位置进行定位
相对于出生位置进行定位
position: relative;
left:100px;
top:100px;

3：固定定位
脱离原来位置进行定位
相对于有定位父级定位，没有的话相对于文档定位
position: fixed;
left:100%;
top:100%;
100%相对于有定位的父级的100%；

浮动模型

float:left/right/both;
1：产生浮动流
2：产生bfc的元素可以看见浮动流
3：文本可以，有padding属性或文本内容的inline标签可以

形成 BFC 的条件
1、浮动元素，float 除 none 以外的值；
2、绝对定位元素，position（absolute，fixed）；
3、display ：inline-block；
4、overflow 除了 visible 以外的值（hidden，auto，scroll）

思考测试总结

解决方法：
（1）为父盒子设置border，为外层添加border后父子盒子就不是真正意义上的贴合 （可以设置成透明：border：1px solid transparent）。
（2）为父盒子添加overflow：hidden；
（3）为父盒子设定padding值；
（4）为父盒子添加position：fixed；
（5）为父盒子添加 display：table；
（6）利用伪元素给子元素的前面添加一个空元素
